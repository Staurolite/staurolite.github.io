    基本语法

    格式

    选择器{ 属性:值;} , “{ 属性:值;}” 称为声明，“属性:值” 称为样式，样式间有分号分隔

    p { fonnt-size:12px;}

    注释代码

    /* ... */

    插入方式

    内联式，如

    <p style="color:red"> the color of text is red <p>

    嵌入式，如

    <head>
    <style type="text/css">
        p{
            color:red;
        }
    </style>
</head>

    外部式，如

    <head>
    <link href="style.css" rel="stylesheet" type="text/css" />
</head>

    把 css 代码写一个单独的外部文件中，这个css样式文件以.css为扩展名，在<head>标签内（不是在<style>标签）使用<link>标签将 css 样式文件链接到HTML文件内。注意：

    1. css样式文件名称以有意义的英文字母命名，如 main.css。
    2. rel="stylesheet" type="text/css"是固定写法不可修改。
    3. 标签位置一般写在标签之内。

    优先级 总体原则：就近原则，一般为 内联式 > 嵌入式 > 外部式，嵌入式 > 外部式 的前提是，嵌入式的 style 语句，写在外部式的 link 语句之后。

    选择器

    css样式声明（定义）由两部分组成，选择器{ 样式;} 在{}之前的部分就是“选择器”，“选择器”指明了{}中的“样式”的作用对象，也就是“样式”作用于网页中的哪些元素

    标签选择器

    格式： 标签{样式;}

    标签选择器其实就是html代码中的标签。如右侧代码编辑器中的 <html>、<body>、 <h1>、<p>、<img>。例如下面代码：

    p{font-size:12px;line-height:1.6em;}

    类选择器

    格式：.类名称{样式;}

    第一步：使用合适的标签把要修饰的内容标记起来，如下：

    <span> this is a example</span>

    第二步：使用 class="class name"为标签设置一个类，如下：

    <span class="sty">this is an example</span>

    第三步：设置类选器css样式，如下：

    .sty{color:red;}

    ID选择器

    格式：#ID名称{样式;}

    与 类选择器 类似 第一步：使用 id="ID name" 标签为标签设置一个ID，如下：

    <span id="sty">this is an example</span>

    第二步：设置类选器css样式，如下：

    #sty{color:red;}

    类选择器 和 ID选择器的区别

    相同点：可以用于任何元素 不同点：

    1. 同一个ID名称只能在文档中出现一次；而类名称可以多次出现
    2. 类名称可以用词列表，可以为同一元素设置多个样式，而Id选择器不可以，如：
    .sty1{
    color:red;
}
.sty2{
    font-size:25px;
}
<p> this is an <span clas="sty1 sty2">example</span></p>

    子选择器

    格式：选择器>子元素{样式;}

    在选择器后，用大于符号(>),选择指定标签元素的第一代子元素，为之设置样式，如下：

    .sty1>span{
    color:red;
}
<p class="sty1"> this is an <span>example</span></p>

    包含选择器

    格式：选择器 后代元素{样式;}

    在选择器后，用加入空格,选择指定标签元素的后代所有元素，为之设置样式。与子选择器的区别在于，子选择器作用于第一代后代元素，包含选择器作用于所有后代元素。如下：

    .sty1 li{
    color:red;
}
<ul class="sty1">
    <li>this is a list</li>
        <ul>
            <li>this is a sublist</li>
        </ul>
</ul>

    通用选择器

    格式：* {样式;}

    用星号 * 指定 HTML 中所有标签元素，为之设置样式，如下：

    * {color:red;}

    伪类选择符

    格式：标签+伪类选择符{样式;}

    针对 伪类选择符，一种 HTML 不存在的标签（指示标签的某种状态），设置样式，

    a:hover{color:red;}

    分组选择符

    格式：选择器,选择器{样式;}

    针对多个选择器，用分组选择符 , 分隔，设置同一样式，如下

    h1,span{color:red;}

    属性

    继承性

    CSS 的某些样式具有继承性，它允许样式不仅用于某个特定的 HTML 标签，而且应用于其后代。当时一些样式则不具有继承性。

    权值

    如果同一个元素被设置了不同的 CSS 样式，根据权值来决定，究竟哪一种样式最终被应用。

    权值的规则：

    选择符 权值
    继承 0.1
    标签 1
    类选择符 10
    ID选择符 100

    例子如下：

    p{color:red;} /*Weight:1*/
p span{color:green;} /*Weight:1+1=2*/
.warning{color:white;} /*Weight:10*/
p span.warning{color:purple;} /*Weight:1+1+10=12*/
#footer .note p{color:yellow;} /*Weight:100+10+1=111*/

    层叠

    如果同一个元素有多个 CSS 样式存在，且权重相同，根据 CSS样式的先后顺序，处于最后面的 CSS 样式会被应用。　

    重要性

    使用 !important，人工设置某些样式拥有最高权值，如：

    p{color:red!important;}
p{color:green;}
<p>this is a paragraph</p>

    注意：!important 要写在分号之前

    当网页没有 CSS 样式时，浏览器会按照固有的一套样式来显示网页，而用户也可以在浏览器设置自己的习惯样式，如字号大小，此时的优先级为：浏览器默认的样式 < 网页制作者样式 < 用户自己设置的样式

    CSS盒模型

    转化为块元素

    a{dispaly:block;}

    上述CSS的设置，将内联元素转化为块元素

    块元素特点：

    1 从新一行开始，其后的元素另起一行 1 元素高度、宽度、行高及顶和底边距都可设置 1 元素宽度不设置的情况下，默认是父容器的 100%

    转化为内联元素

    div{display:inline;}

    上述CSS的设置，将块元素转化为内联元素

    内联元素特点：

    1 和其他元素在同一行上 1 元素的高度、宽度及顶部和底部边距不可设置 1 元素的宽度就是它包含的文字或图片的宽度，不可改变

    转化为内联块元素

    p{display:inline-block;}

    inline-block 元素特点： 1 和其他元素在同一行上 1 元素高度、宽度、行高及顶和底边距都可设置

    盒子模型元素

    Padding 内边距，Margin 外边距，Border 边框，三者还可以分别在上下左右四个方向设置

    边框

    边框可以设置三个属性：粗细、样式、颜色。

    div{border: 2px solid red;}
div{
    border-width:2px;
    border-style:solid;
    border-color:red;
}

    border-width（边框宽度）粗细设置：thin|medium|thick，还有常用的像素（px）； border-style（边框样式）常见样式：dashed（虚线）|dotted（点线）|solid（实线）； border-color（边框颜色）可设置为十六进制颜色或者颜色名称，如 #888，red；

    边框的四周上下左右边框可以分别设置：

    div{
    border-top:1px solid red;
    border-bottom:1px solid red;
    border-left:1px solid red;
    border-right:1px solid red;
}

    填充

    填充：元素内容与边框之间的距离，填充分为 上，右，下，左（顺时针）,可以分开写或者按顺序写，以及包含三种可以缩写的特殊情况1.四周都一样，2.上下一样、左右一样，3.左右一样。

    div{
    padding-top:10px;
    padding-right:20px;
    padding-bottom:30px;
    padding-left:40px;
}

div{padding:10px 20px 30px 40px;}

div{padding:10px;}

div{padding:10px 20px;}

div{padding:10px 20px 30px}

    边界

    填充：元素内容与其他元素之间的距离，填充分为 上，右，下，左（顺时针）,可以分开写或者按顺序写，以及包含三种可以缩写的特殊情况1.四周都一样，2.上下一样、左右一样，3.左右一样。

    div{
    margin-top:10px;
    margin-right:20px;
    margin-bottom:30px;
    margin-left:40px;
}

div{margin:10px 20px 30px 40px;}

div{margin:10px;}

div{margin:10px 20px;}

div{margin:10px 20px 30px}

    宽度和高度

    盒子模型的宽度是 左边界+左边框+左填充+内容宽度+右填充+右边框+右边界,高度也同理。

    CSS 布局模型

    元素的三种布局模型：

    1. 流动模型（Flow）
    2. 浮动模型（Float）
    3. 层模型（Layer）

    流动模型

    流动模型（Flow）是默认的网页布局模式，2个典型特征：

    1. 块元素会内自上而下按顺序垂直延伸分布显示；
    2. 内联元素会从左到右水平分布显示；

    在默认状态下，块状元素的宽度都为100%，实际上，块状元素都会以行的形式占据位置；而内联元素不会霸占整行。

    浮动模型

    浮动模型（Float）可以是两个元素并排显示

    div{
    width:200px;
    height:200px;
    border:2px red solid;
}
#div1{float:left;}
#div2{float:right;}

<div id="div1"></div>
<div id="div2"></div>

    CSS浮动(float,clear)通俗讲解

    层模型

    层模型（Layer），通过一组定位（positioning）属性来支持布局到达精准定位的目的。 层模型的三种形式：

    1. 绝对定位(position: absolute)
    2. 相对定位(position: relative)
    3. 固定定位(position: fixed)
    绝对定位

    绝对定位，需要设置position:absolute。这条语句的作用将元素从文档流中拖出来，然后使用left、right、top、bottom属性在其最接近的一个具有定位属性的父元素进行绝对定位。如果不存在这样的父元素，则相对于body元素，即相对于浏览器窗

    div{
    width:200px;
    height:200px;
    border:2px red solid;
    position:absolute;
    left:100px;
    top:50px;
}
<div id="div1"></div>
    相对定位

    相对定位，需要设置position:relative。语句首先按static(float)方式生成一个元素(并且元素像层一样浮动了起来)，通过left、right、top、bottom属性确定元素相对于以前位置的偏移，但是偏移前的位置还是会保留。

    #div1{
    width:200px;
    height:200px;
    border:2px red solid;
    position:relative;
    left:100px;
    top:50px;
}
<div id="div1"></div>
    固定定位

    固定定位，需要设置position: fixed。与绝对定位定位方式类似，但元素的坐标是是相当于浏览器视窗本身。浏览器视窗本身是固定的，元素不会随滚动条滚动而变化，除非移动浏览器的位置，或改变浏览器窗口的显示大小。因此固定定位的元素会始终位于浏览器窗口内视图的某个位置，不会受文档流动影响，这与background-attachment:fixed;属性功能相同。

    #div1{
    width:200px;
    height:200px;
    border:2px red solid;
    position:fixed;
    left:100px;
    top:50px;
}
<p>contentcontentcontentcontent</p>
    相对定位与绝对定位的组合

    绝对定位相对于父元素块而不是body元素块定位的设置：

    1. 参照定位的父元素块设置为相对定位（relative）：
    2. 需要定位的元素设置为绝对定位（absolute），通过四向的偏移微调
    #box1{
    width:200px;
    height:200px;
    position:relative;
}   
#box2{
    position:absolute;
    top:20px;
    left:30px;         
}
<div id="box1">father
    <div id="box2">son</div>
</div>

    CSS 代码简写

    颜色值缩写

    当设置的颜色是16进制的色彩值时，如果每两位的值相同，可以缩写一半

    p{
    color:#000000;
    color:#000;
    color:#336699;
    color:#369;
}

    字体缩写

    网页字体设置代码

    body{
    font-style:italic;
    font-variant:small-caps; 
    font-weight:bold; 
    font-size:12px; 
    line-height:1.5em; 
    font-family:"宋体",sans-serif;
}

    缩写为：

    body{
    font:italic small-caps bold 12px/1.5em "宋体",sans-serif;
}

    要点：

    1. 使用简写时，至少要指定 font-size 和 font-family 属性，其他属性如未指定使用默认值。

    2.在缩写时 font-size 和 line-height 中间要加 "/" 斜杠。

    针对中文网站，下面的代码比较常用：

    body{
    font:12px/1.5em "宋体",san-serif;
}

    颜色与长度

    颜色值

    1. 英文命令
    2. RGB颜色,值可以是0~255的整数，或者0%~100%的百分数
    3. 十六进制颜色，值为0~255变成十六进制的00-ff
    p{
    color:red;
    color:rgb(133,45,200);
    color:rgb(20%,33%,25%);
    color:#00ffff;
}

    长度

    1. 像素px，90像素 = 1英寸，和显示器实际像素值有关
    2. em，1em = font-size，以父元素的font-size为准
    3. 百分比，行高为font-size的百分比
    p{font-size:14px;line-height:130%} 
span{font-size:0.8em}

    设置技巧

    行内元素居中

    水平居中，通过父元素设置text-align:center来实现

    div{text-align:center}
<div>content</div>

    定宽块元素居中

    定宽块元素居中，通过设置左右margin:auto来实现,上下margin可以随意设置

    div{
    border:2px solid red;
    width:500px
    margin:20px auto /*margin-left marigin-right set for auto*/
}
<div>content</div>

    不定宽块元素居中

    1. 加table标签
    2. 设置display:inline
    3. 设置position:relative 和left:50%
    table标签法
    1. 在需要设置的块元素外面加入一个table标签（包括 ）
    2. 为table设置“左右margin居中”
    table{margin:0 auto;}

<table>
    <tbody>
        <tr>
            <td>
                <div>content</div>
            </td>
        </tr>
    </tbody>
</table>
    设置为内联元素
    1. 把块元素设置为内联元素，display = inline
    2. 然后把父元素设置为 text-align:cneter
    <style>
    .container{text-align:center;}
    .container ul{
                list-style:none;margin:0;padding:0;
                display:inline;
                }
    .container li{
                margin-right:8px;
                display:inline; 
                }
</style>

<div class="container">
    <ul>
        <li><a href="#">1</a></li>
        <li><a href="#">2</a></li>
        <li><a href="#">3</a></li>
    </ul>
</div>
    设置布局模型
    1. 父元素设置 float:left
    2. 父元素设置 position:relative 和 left:50%
    3. 子元素设置 position:relative 和 left:-50%
    .wrap{
    float:left;
    position:relative;
    left:50%;
    clear:both;
}

.wrap-center{
    background:#ccc;
    position:relative;
    left:-50%;
}

<div class="wrap">
    <div class="wrap-center">content</div>
</div>

    垂直居中

    单行文本垂直居中

    单行文本垂直居中，通过设置父元素 height 和 line-height 高度一致来实现

    .container{
    height:100px;
    line-height:100px;
}
<div class="container">
    content
</div>
    多行文本垂直居中
    1. 插入table标签（包括tbody、tr、td标签），设置vertical-align:middle。此为td标签默认设置，故不需要显示设置；
    2. 设置块元素display:table-cell，及vertical-align:middle，需浏览器支持

    table标签法

    .wrap{height:500px;background:#ccc}
<table><tbody><tr><td class="wrap">
<div>
    <p>content</p>
    <p>content</p>
    <p>content</p>
    <p>content</p>
    <p>content</p>
</div>
</td></tr></tbody></table>

    table-cell法

    .container{
    display:table-cell;
    vertical-align:middle;
}
<div class="container">
    <div>
        <p>content</p>
        <p>content</p>
    </div>
</div>

    隐性改变display类型

    设置以下两个语句，元素自动以 display:inline-block 方式显示，可以设置元素的 width 和 height 且默认宽度不占满父元素。

    1. position:absolute
    2. float:left 或 float:right 
    .container{
    position:absulte;
    width:200px;
}

<div class="container">
